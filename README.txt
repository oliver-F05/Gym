GYM TRACKER V2

This is the redesigned app matching the approved orange-and-black mockup.

TEST ON WINDOWS
1. Extract the ZIP.
2. Open PowerShell in the extracted folder.
3. Run:
   py -m http.server 8000
4. Open http://localhost:8000

IPHONE
Host the folder on GitHub Pages, Cloudflare Pages, Netlify or Vercel.
Open the hosted URL in Safari, tap Share > Add to Home Screen.

DATA
All workouts are stored locally on the device/browser.
Use Settings > Backup regularly.
